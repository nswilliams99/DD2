# Employee Access Guide - Payton

## Getting Started

### GitHub Repository Access
- Repository: https://github.com/nswilliams99/dumpster_diverz_website
- You should have access to view and clone the repository
- All documentation is in the `/docs` folder

### Supabase Access Needed
You'll need access to the Supabase dashboard for:
- **Storage**: Upload images to website_pics bucket
- **Database**: Verify content in tables
- **Logs**: Monitor performance and errors

**Ask for access to:**
- Supabase project dashboard
- Storage bucket permissions (website_pics)
- Database read access for content verification
